# 车企周边好物商城 · 智能驾驶舱原型 Demo

## 快速开始 🚀

1. 直接双击 `index.html` → 在浏览器里打开即可查看完整的交互式驾驶舱演示
2. 无需安装任何软件，无需启动服务器
3. 需要联网加载 ECharts 图表库（首次打开时自动下载，之后缓存）

> **提示**：如需离线使用，可下载 [ECharts](https://echarts.apache.org/zh/download.html) 压缩包，将 `echarts.min.js` 放在与 `index.html` 同级目录，然后修改 HTML 第 7 行为 `<script src="echarts.min.js"></script>`。

## 页面说明

| 页面 | 功能描述 |
|------|----------|
| 🌐 全局态势 | 核心 KPI 卡片（销售额/毛利/利润/T3完成率/提车量/渗透率）、月度趋势图、渠道分布、告警摘要 |

| ⚡ 行动中枢 | 异常告警列表、月中预测、结构变化追踪 |

## 演示话术（给业务方演示时使用）

### 3 分钟主流程

```
1. 打开页面 → "当前T3完成率60%，月中预测本月只能达到72%"
2. 查看告警摘要 → "系统自动识别到T1N渗透率连续3月下降"
3. 切到归因分析 → "量价分解显示：价的问题贡献了63%的缺口"
4. 维度下钻 → "车生长贡献了75%的差距"
5. 切到行动中枢 → "这些告警不用等月报，实时可见"
6. 总结：驾驶舱能做到月中预测+自动告警+问题归因，不只是看目标完成率
```

### 核心价值点

- **Level 1 · 监控**：实时目标完成率，不用等月底报表
- **Level 2 · 预警**：月中就能预测月底完成率，提前介入
- **Level 3 · 洞察**：趋势异常自动识别，不用人工翻数据
- **Level 4 · 归因**：量价分解 + 维度贡献，直接找到问题根因

## 数据说明

Demo 使用模拟数据（内嵌在 HTML 文件中），包含以下故事线：

| 故事线 | 描述 |
|--------|------|
| T1N 渗透率慢性恶化 | 连续 4 月下降：35% → 33% → 30% → 28% |
| 车生长占比结构变化 | 从 45% 缓慢降到 36%，月报看不出来，趋势图一眼可见 |
| 华南大区区域异常 | T3完成率从 85% 跌至 52%，其他大区基本正常 |
| APP 商城逆势上升 | 占比从 15% 增长到 22%，是值得关注的亮点渠道 |

所有数据均为演示用途，不代表真实业务数据。
