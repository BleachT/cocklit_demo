# 车企周边好物商城 · 智能驾驶舱原型 Demo

## 快速开始 🚀

1. 直接双击 `index.html` → 在浏览器里打开即可查看完整的交互式驾驶舱演示
2. 无需安装任何软件，无需启动服务器
3. 需要联网加载 ECharts 图表库（首次打开时自动下载，之后缓存）

> **提示**：如需离线使用，可下载 [ECharts](https://echarts.apache.org/zh/download.html) 压缩包，将 `echarts.min.js` 放在与 `index.html` 同级目录，然后修改 HTML 第 7 行为 `<script src="echarts.min.js"></script>`。

## 演示话术（给业务方演示时使用）

### 3 分钟主流程

```
1. 打开全局态势页面 → "GMV 连续3日下滑，当前退货率6.8%并恶化了1.2pp"
2. 点击告警 "GMV 连续3日下滑>5%" → 跳到归因分析
3. 看瀑布图 → "量的问题占40%，价的问题占60%，旗舰品类客单价下跌是主因"
4. 回到全局态势 → "供应商A-03延迟率42%，已影响1,280笔订单"
5. 切到行动中枢 → "系统已自动推荐3个行动方案，可直接审批执行"
6. 总结：驾驶舱实现了感知异常→定位根因→推荐行动的完整闭环
```

### 核心价值点

- **感知（Sense）**：实时 KPI 监控 + 自动异常告警，不用等日报
- **理解（Understand）**：AI 归因分析，直接告诉你问题出在哪个维度
- **决策（Decide）**：系统推荐行动方案并高亮最优解
- **行动（Act）**：一键执行 + 结果追踪，形成业务闭环

## 数据说明

Demo 使用模拟数据（内嵌在 HTML 文件中），包含以下业务故事线：

| 故事线 | 描述 |
|--------|------|
| GMV 三日下滑预警 | 旗舰品类客单价下跌 + 旗舰品销售萎缩，归因已定位 |
| 供应商 A-03 延迟危机 | 发货延迟率 42%，1,280 笔订单受影响，退款风险 ¥86K |
| 车载香薰库存告急 | 3 个 SKU 预计 2 天内断货，需紧急补货 |
| NPS 评分下滑 | 4 月下降 4.1 分，负面评价集中在"物流速度" |

所有数据均为演示用途，不代表真实业务数据。
