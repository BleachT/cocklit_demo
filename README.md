# 车企周边好物商城 · 智能驾驶舱 Demo

基于 **Palantir Foundry** 核心理念设计的驾驶舱 Demo，用于车企周边好物商城的数据决策支持。

## 核心理念

借鉴 Palantir 的两大核心技术：

| 理念 | 实现 |
|------|------|
| **本体论建模（Ontology）** | 以业务对象（商品、供应商、订单、用户等）为中心组织数据，通过语义关系连接，形成知识图谱 |
| **归因分析（Attribution）** | 异常指标自动归因，展示贡献因子排名、因果传导链路、根因瀑布图 |

## 五大视图

1. **🌐 全局态势** — KPI 总览 + 异常告警 + 多维图表
2. **🔍 归因分析** — GMV 下滑根因分析、因果链路追溯、时序回溯
3. **🕸 本体图谱** — 交互式知识图谱，可视化对象关系网络
4. **📦 对象详情** — 供应商 360° 视图，属性 + 关联对象 + 趋势
5. **⚡ 行动中枢** — 感知→理解→决策→行动闭环，一键触发业务操作

## 快速启动

直接在浏览器中打开 `index.html` 即可，无需任何构建步骤。

```bash
open index.html
```

## 技术栈

- **ECharts 5** — 数据可视化图表
- **纯 HTML/CSS/JS** — 无需构建工具，开箱即用
- **暗色主题** — Palantir 风格高对比度界面
