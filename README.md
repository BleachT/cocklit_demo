## 快速开始

```
1. 下载/克隆本仓库
2. 进入 prototype/ 文件夹
3. 双击 index.html → 浏览器打开
4. 完整的交互式驾驶舱就在眼前 ✅
```

## 驾驶舱功能概览

- **全局态势**：6 大 KPI 卡片 + 月度趋势 + 渠道分布 + 告警摘要
- **归因分析**：量价瀑布图 + 维度贡献排行
- **本体图谱**：业务对象关系可视化
- **对象详情**：产品/渠道/大区数据钻取
- **行动中枢**：异常告警列表 + 月中预测

## 如何在本地查看

**方法一：直接下载（推荐）**
1. 点击页面右上角绿色 `<> Code` 按钮
2. 选择 `Download ZIP`
3. 解压后双击 `prototype/index.html`

**方法二：克隆仓库**
```bash
git clone https://github.com/BleachT/cocklit_demo.git
cd cocklit_demo
open prototype/index.html   # Mac
# 或双击文件夹中的 prototype/index.html（Windows）
```
