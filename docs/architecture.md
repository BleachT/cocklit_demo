# 车企车周边好物商城 · 数据驾驶舱技术方案

> **定位**：面向运营、商品、供应链、市场团队的一站式数据驾驶舱，帮助用户快速发现经营问题、理解根因、指导下一步行动。

---

## 1. 驾驶舱功能模块

| 模块 | 核心功能 | 关键指标 |
|------|---------|---------|
| **总览大盘** | 全局经营健康度快照，异常红绿灯 | GMV、订单量、新增用户、客单价、退货率 |
| **销售分析** | 品类/渠道/区域/品牌多维拆解 | 各维度 GMV、增长率、客单价 |
| **库存监控** | 实时库存健康评分，缺货/积压预警 | 可售天数、安全库存达成率、库存周转率 |
| **用户洞察** | 转化漏斗、留存队列、LTV 分析 | 购买转化率、次周留存率、4周留存率、复购率 |
| **问题发现 & 行动建议** | 自动告警 + 根因 + 结构化行动建议 | 告警数、处理率、问题闭环周期 |

## 3. 问题发现机制（核心设计）

### 3.1 告警规则分层

| 层级 | 方法 | 示例 |
|------|------|------|
| **阈值告警** | 固定阈值 / 百分比变化 | GMV 环比下滑 > 15% → 严重告警 |
| **统计告警** | 3σ 规则 / IQR 箱线图 | 退货率超出历史均值 ±3σ |
| **趋势告警** | 滑动窗口线性回归 | 连续 5 日新增用户下滑 |
| **复合告警** | CEP（Flink）多条件组合 | GMV 下滑 + 流量下滑 + 转化率下滑 同时触发 |

### 3.2 根因分析框架

```
告警触发
   │
   ├── 自动分维度下钻（品类/渠道/区域）找主贡献维度
   ├── 关联因子检测（流量、转化率、客单价哪个先变）
   ├── 外部因子匹配（节假日、促销、竞对、天气）
   └── 输出：「主要由 X 驱动，同比/环比贡献 Y%」
```

### 3.3 行动建议生成

- **规则库**：预定义场景 → 行动 SOP 映射（当前实现）
- **LLM 增强**（生产建议）：将指标异常 + 历史处置记录 + 业务知识库 → RAG → GPT/本地大模型 → 生成结构化行动建议，指定责任团队

## 4. 数据指标体系

### 4.1 销售指标

| 指标 | 计算口径 | 更新频率 |
|------|---------|---------|
| GMV | 支付成功订单金额 | 实时（1分钟） |
| 订单量 | 支付成功订单数 | 实时 |
| 客单价 | GMV / 订单量 | 实时 |
| 退货率 | 退货订单数 / 支付订单数 | T+1 |
| 品类贡献度 | 品类GMV / 总GMV | T+1 |

### 4.2 用户指标

| 指标 | 计算口径 | 更新频率 |
|------|---------|---------|
| 购买转化率 | 支付用户数 / 访问用户数 | T+1 |
| 次周留存率 | 第2周活跃用户 / 入组用户 | T+1 |
| 复购率 | ≥2次购买用户 / 总购买用户 | T+1 |
| 购物车放弃率 | (加购-购买) / 加购 | 实时 |

### 4.3 库存指标

| 指标 | 计算口径 | 更新频率 |
|------|---------|---------|
| 可售天数 | 当前库存 / 近7日日均销量 | 实时（推仓即更新） |
| 安全库存达成率 | 实际库存 / 安全库存阈值 | 实时 |
| 库存周转率 | 销货成本 / 平均库存额 | T+1 |
| 积压金额 | 可售天数 > 90天的SKU库存价值 | T+1 |

## 5. 技术选型说明

| 组件 | 选型 | 理由 |
|------|------|------|
| **消息队列** | Apache Kafka | 高吞吐、顺序消息、支持回放 |
| **CDC** | Debezium | 无侵入捕获 MySQL/PG binlog |
| **实时计算** | Apache Flink | CEP + 状态管理 + 精确一次语义 |
| **离线计算** | Apache Spark | 生态成熟，与 Hive 集成好 |
| **数仓格式** | Apache Iceberg | 支持 ACID、Schema Evolution、分区裁剪 |
| **OLAP** | ClickHouse | 列式存储，亿级数据秒级查询 |
| **缓存** | Redis | 实时指标缓存，降低 ClickHouse 压力 |
| **可视化** | Streamlit + Plotly | 快速迭代，交互性强 |
| **埋点** | 神策 / GrowingIO | 行为数据采集，漏斗/留存分析 |
| **告警推送** | 钉钉/企微 Webhook | 团队即时通知 |
